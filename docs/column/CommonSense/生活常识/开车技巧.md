# 倒车停车与车辆控制技巧

::: tip 常用技巧：“灯镜重合”

“灯镜重合”是一个在**倒车或侧方停车**时使用的**视觉参考点**。它指的是从驾驶员视角看，本车的**外侧后视镜**（通常是右侧）与相邻车辆的**尾灯**（或特定车身部位）在视觉上**对齐**的瞬间。

**常用于：**

- 侧方位停车
- 倒车出库（两侧有车时）
- 防止车头刮蹭

:::

## 🚗 侧方位停车（标准方法）

1.  **初始位置**：与右侧车位前车平行，横向间距约 **0.5米**。
2.  **开始倒车**：**向右打死方向盘**，开始倒车。
3.  **观察左后视镜**：当从左后视镜中看到**后方车辆整个车头完全露出**时，**将方向盘回正**，继续直线倒车。
4.  **关键参考点（任选其一）**：
    - 观察右前门把手与路牙即将重合。
    - 或者，观察本车右侧后视镜与右车尾灯重合（即“灯镜重合”）。
5.  **入库**：到达参考点后，**迅速向左打死方向盘**，慢慢倒车入位。车正后回正方向盘。

## 🔲 90度倒车入库（垂直入库，一把打死）

1.  **准备摆位**
    - 将车辆开到与目标车位**平行**的位置。
    - 车身与车位线（或旁车）保持约 **1.5米** 的横向距离。
    - 车辆的**B柱**大约与隔壁车位的**远端车辆（或库角）** 对齐。
2.  **开始倒车**：向车位方向（向右或向左）将方向盘打死开始匀速倒车。
3.  **关键观察点**
    - **观察内侧后视镜**：主要关注本车与近端库角（或旁车车头）的距离，确保不会剐蹭。
    - **观察外侧后视镜**：当车身与两侧的车位线或车辆即将**平行**时，就是回正方向的时机。
4.  **回正与调整**：看到车身将正时，迅速将方向盘回正。

## ◢ 45度倒车入库（车位较窄时斜向入库 - 常用）

1.  **准备摆位**：将车辆开到与目标车位呈**约45度角**的位置。
2.  **开始倒车**：挂入倒挡，先向车位方向打死方向，开始倒车。
3.  **关键观察点**
    - 重点观察**内侧（靠近车位侧）后视镜**，确保车尾不会刮蹭到库角或旁车。
    - 同时观察**外侧后视镜**，当车身通过大约45度角，即将与车位线平行时，准备回方向。
4.  **回正与调整**：
    - 倒车时发现车尾不能入库时，挂前进档向前走一点，在继续打死方向倒车。
    - 当车身与车位线基本平行时，**将方向盘完全回正**。
    - 直线倒车完成入库，或进行微调。

## 🔄 短距侧方移位（停歪或太远时调整）

当侧方停车后发现车身离路牙较远或不正时，可通过此方法短距离调整，使车尾靠近路牙。

- **核心原理**：每次倒车都**向左打死**，每次前进都**向右打死**，快接近前车时再反方向调整。
- **操作循环**：
  1.  **倒车**：挂倒挡，**向右打死**方向倒车。
  2.  **前进**：挂前进挡，**向右打死**方向前进。
  3.  **临近前车**：快接近前车时，**向左打死**方向调整。
  4.  **停车**：停车时，务必**回正方向盘**。

## 📏 停车距离判断

### 1. 跟车停车距离（等红灯时）

- **判断方法**：在驾驶位，当**刚好看不到前车后轮胎与地面接触点**（或车头与前车后保险杠下沿重合）时停车。
- **实际距离**：此时距前车约为 **2米** 的安全距离。

### 2. 作为头车距停止线距离

- **判断方法**：在驾驶位，观察左侧后视镜下沿，当其**与前方停止线（或路面标线）重合**时停车。
- **实际距离**：此时车头距停止线约 **30厘米**。

### 3.前方有障碍物时停车距离

- **判断方法-1**： 当车子的前机舱盖与障碍物底部重合时，车头距离障碍物大约2.5米，此时打满方向转弯，车头不会刮蹭。
- **判断方法-2**： 当A柱夹角，与前方障碍物底部重合时，车头距离障碍物大约2米，这是打满方向转弯的极限点位，再近就会刮蹭。

## 🅿️ 车头顺进侧方停车

当车位前方空间较大时，可采用车头先进的方式。

1.  **切入时机**：当本车**右侧B柱或后视镜**越过旁车车头（或车尾）时。
2.  **第一次打方向**：**向右打一圈方向**，向车位内前进。
3.  **第一次回正**：当道路边缘（路牙）出现在前挡风玻璃**右侧约2/3位置**时，**回正方向盘**。
4.  **第二次调整**：紧接着**向左打一圈方向**，调整车身。
5.  **摆正停车**：待车身即将摆正时，**回正方向盘**，停车。

- **提示**：前方空间越长，第一次切入时方向盘打的角度可以越小（如半圈）。

## ↩️ 倒车出车位处理（两侧有车）

1.  **直倒观察**：保持方向盘正直向后倒车，同时观察两侧后视镜。
2.  **关键点**：当本车后视镜（或B柱）与旁边车辆的**尾灯部位重合**（灯镜重合）时，就是开始打方向的时机。
3.  **转向倒出**：向希望驶出的一侧**打死方向盘**，继续倒车。
4.  **观察前方**：当从车内能看到**前车（原侧前方车辆）的整个保险杠**时，**回正方向盘**。
5.  **完成驶出**：然后**向另一侧打死方向盘**，即可完成转向，驶离车位。

## 🛞 方向控制精细技巧

### 1. 半把方向法（靠边30cm）

- **操作**：先向右打 **180度**（半圈），观察前机盖，当道路边缘位于前挡风玻璃**右侧约2/3**时，向左回正（180度）。接着向左打180度调整，待车身即将摆正时，向右回正180度。
- **结果**：车身右侧距路边约 **30厘米**。

### 2. 立把方向法（靠边30cm）

- **操作**：先向右打 **90度**（立角），观察前机盖，当道路边缘移动到前挡风玻璃**正中间**时，向左回正（90度）。接着向左打90度调整，待车身即将摆正时，向右回正90度。
- **结果**：车身右侧距路边约 **30厘米**。

## ⚙️ 掉头方向盘操作技巧（狭窄路段）

- **开始掉头**：先迅速向左打**一整圈（360度）**。
- **观察调整**：若发现转向不足，**虚握**方向盘再将剩余的**半圈（180度）** 打完至极限。
- **回正时机**：当车辆掉头过半，车头开始对准新车道时，**先回半圈（180度）**。
- **完成回正**：待车身完全进入新车道并摆正后，**再回最后一整圈（360度）**，方向盘即完全回正。
