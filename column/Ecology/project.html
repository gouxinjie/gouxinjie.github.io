包管理工具

  • npm

    Node.js官方包管理工具，拥有最大的JavaScript包生态系统

  • yarn

    Facebook开发的包管理工具，提供确定性安装和离线模式

  • 热门

    pnpm

    高效包管理器，通过硬链接节省磁盘空间，保持node_modules结构

  • 热门

    nvm

    Node.js版本管理工具，支持多版本切换

  • 热门

    fnm

    基于Rust实现的Node.js版本管理工具，支持多版本切换，快速安装和切换

  • corepack

    Node.js内置包管理器管理器，统一yarn/pnpm体验

  • Bun

    新兴的JavaScript运行时，内置快速包管理器

构建工具(工程打包)

  • webpack

    模块化打包工具，支持代码分割、懒加载等高级功能

  • 热门

    Rollup

    ES模块打包器，适合库开发，支持Tree-shaking

  • 热门

    esbuild

    极速JavaScript打包器，基于Go语言开发

  • 热门

    vite

    下一代前端构建工具，开发环境基于原生ESM，生产使用Rollup

  • 热门

    Turbopack

    Webpack作者开发的Rust版打包工具，速度极快

  • 热门

    rsbuild

    基于Rust的前端构建工具，速度极快

  • Rome

    一体化前端工具链，包含编译、打包、lint等功能

  • Parcel

    零配置打包工具，支持多类型资源处理
